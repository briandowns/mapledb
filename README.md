# MapleDB

MapleDB is a simple key/value store module with a very simple API. 

* get(key: String) -> String
* set(key: String, value: Any) -> Nil
* del(key: String) -> Nil
* contains(key: String) -> Boolean
* keys() -> List<String>
* values() -> List
* items() -> List<List>
